## 📦 **For Native App Installation (APK):**

### **Step 1: Build APK**
```bash
cd client
npm run build
npx cap open android
```

### **Step 2: In Android Studio**
1. Wait for Android Studio to open
2. Click **Build** → **Build Bundle(s)/APK(s)** → **Build APK(s)**
3. Wait for build to complete
4. Click **"locate"** to find your APK file

### **Step 3: Install on Phone**
1. Copy the APK file to your phone (USB/email/cloud)
2. Enable "Install from unknown sources" in phone settings
3. Tap the APK file to install
4. Your N-Queens game is now a native Android app!

## 🔧 **Troubleshooting:**

**If the URL doesn't work on your phone:**
1. Make sure both devices are on the same WiFi
2. Try: `http://localhost:5173` (if on same computer)
3. Check Windows Firewall settings
4. Try restarting both servers
